◆ readData()

template<typename T >
int circularBuffer< T >::readData ( T *  dataPtr)
inline

Copies a data point from the buffer and increments the internal read pointer without deleting the data. It starts by copying the oldest data in the buffer at returns 1. In subsequent calls, it will copy more recent data until all data is copied. Then, it will reset the internal read pointer to the oldest data point and return 0.

Parameters
dataPtrPointer to the location to which you want the data point to be copied.
Returns
Returns 1 if more data can be copied. Returns 0 if no more data can be uniquely copied. Returns -1 if the buffer is empty.

◆ readRemoveData()

template<typename T >
int circularBuffer< T >::readRemoveData ( T *  dataPtr)
inline

Copies a data point from the buffer and removes it from the buffer. Also increments the internal read pointer. It starts by copying the oldest data in the buffer at returns 1. In subsequent calls, it will copy more recent data until all data is copied. At this point, it returns 0.

Parameters
dataPtrPointer to the location to which you want the data point to be copied.
Returns
Returns 1 if more data can be copied and then deleted. Returns 0 if all data has been copied and deleted. Returns -1 if the buffer is empty. Returns -2 if a no-delete read operation is in progress.
